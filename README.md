# ScalaMeter
Using `org.scalameter._` package on top of Gradle Kotlin DSL project...

_usage_

```bash
./gradlew ; java -jar ./build/libs/*-all.jar
```

_output example_

```bash
executed "(0 until 1000000).toArray.reverse" in: 50.537401 ms
```
